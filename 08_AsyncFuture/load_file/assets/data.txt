Задание 2. ( load file )

Что нужно сделать

1. Напишите функцию загрузки файла из папки assets.
2. Добавьте в приложение виджет TextField и стилизуйте его, как
показано на скриншоте.
Критерии оценки
Кнопка одного размера с текстовым полем. В текстовое поле
вводится имя файла, и если такой файл существует в папке assets,
то контент файла отображается на экране, если такого файла не
существует, то отображается текст «файл не найден». Формат
файлов — .txt.

Рекомендации и советы
Функцию загрузки файла можно взять из задания 1.
